### Note on `.env` loading
The project does **not** use `dotenv`. To run locally, you must either:
- Export vars manually: `export $(cat .env.local | xargs) && node server.js`
- Or install dotenv and add `require('dotenv').config()` to `server.js`
- Or use `vercel dev` (which loads `.env.local` automatically)
